本周专题：LCA（最近公共祖先）
学习目标：理解LCA的原理，掌握LCA的三种求法
学习要求：要求题目按规定的方法完成

ps：LCA（Least Common Ancestors），即最近公共祖先，是指在有根树中，找出某两个结点u和v最近的公共祖先。

模板题
Nearest Common Ancestors

要求用Tarjan算法(离线)，RMQ(在线)，倍增(在线)三种方法来写这道题
推荐的学习博客：对LCA、树上倍增、树链剖分（重链剖分&长链剖分）和LCT（Link-Cut Tree）的学习
